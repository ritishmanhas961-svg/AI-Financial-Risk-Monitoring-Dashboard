# AI Financial Risk Monitoring and Fraud Detection Dashboard

## Project Overview

This project focuses on identifying financial risks and fraudulent activities using Power BI analytics. The dashboard provides insights into revenue performance, fraud trends, loan risk levels, department-wise financial exposure, and geographical fraud hotspots.

The objective is to help organizations monitor financial health, detect suspicious transactions, and improve decision-making through visual analytics.

---

## Business Problem

Financial institutions and organizations often face challenges such as:

- Fraudulent transactions
- Delayed loan approvals
- High-risk borrowers
- Revenue leakage
- Regional fraud concentration

This dashboard helps management identify potential risks and take preventive actions.

## Dataset Structure

| Column Name | Description |
|------------|-------------|
| Loan_ID | Unique Loan Number |

| Revenue | Total Revenue |
